Le fichier tsconfig.json

TypeScript vérifie si un fichier de configuration tsconfig.json existe et utilise les paramètres qui y sont définis.

Avantage : Plus besoin de s'inquiter des problèmes de compatibilité entre les versions

Comment obtenir le fichier tsconfig.json :

  • Le créer manuellement à la racine du projet et le compléter selon les besoins
  • Utilisez la commande : 
    tsc --init
  • Utiliser un générateur en ligne, par exemple : tsconfig.json
  • Une fois le fichier tsconfig.json en place, vous pouvez utiliser la commande suivante pour écouter le projet 
    tsc -w
