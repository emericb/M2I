Installation de typeScript

  • Installer Node.js
  • Vérifier l'Installation de Node.js avec la commande : 
    node -v
  • Installer TypeScript en global avec la commande 
    npm install -g typescript
  • Vérifier l'Installation de TypeScript avec la commande : 
    tsc -v

Mon premier script TypeScript

Pour vérifier que typeScript est correctement installé

  • Ajouter une balise script avec en src un fichier .js
  • Créer un fichier .ts, puis dans la console exécuter la commande : 
    tsc mon-fichier.ts

Utilisation de tsc avec l'option -w

  • Avec l'option -w ou --watch, vous pouvez préciser le ou les fichier à surveiller
  • Utilisez ctrl + c dans le terminal pour arrêter la surveillance.
